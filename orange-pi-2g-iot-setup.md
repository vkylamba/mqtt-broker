# Orange Pi 2G IoT Setup
### Image to use:

1. Raspbian

### Debug port console:
1. baudrate: 115200

### Resize SD card file system:
  1. fs_resize

### Setup python:
  1. Install python3: apt-get update && apt-get install python3-pip
  2. Install virtual env: pip3 install virtualenv
  3. Create virtual env: virtualenv penv -p python3
